སྐབས་དྲུག་པ། མཐར་གྱིས་སྦྱོར་བའི་སྦྱོར་ཊཱིཀ

གཉིས་པ་བརྟན་པ་རྒྱུ་འབྲས་ལ་གཉིས། རྒྱུ་མཐར་གྱིས་པ་དང༌། འབྲས་བུ་སྐད་ཅིག་མའོ། །དང་པོ་ ( རྒྱུ་མཐར་གྱིས་པ་ ) ལ་གསུམ། འབྲེལ་བཀོད་པས་མཚམས་སྦྱར། མཚོན་བྱེད་ཀྱི་ཆོས་རྒྱས་པར་བཤད། སྐབས་ཀྱི་གྲངས་བརྗོད་པས་མཇུག་བསྡུ་བའོ། །དང་པོ་ ( འབྲེལ་བཀོད་པས་མཚམས་སྦྱར་ ) ནི། འགྲེལ་པ། རྩེ་མོའི་མངོན་པར་རྟོགས་པ་ཐོབ་པ་སོ་སོ་བ་དང་བསྡུས་པ་ཉིད་དུ་རྟོགས་པའི་དོན་རྣམས་གོ་རིམ་བཞིན་དུ་བཀླགས་ནས་བརྟན་པར་བྱ་བའི་ཕྱིར་རྣམ་པར་སྒོམ་པར་བྱེད་པས་མཐར་གྱིས་པའི་མངོན་པར་རྟོགས་པ། འགྲེལ་པའི་སྦྱོར་ཊཱིཀ རྩེ་སྦྱོར་གྱི་འོག་ཏུ་མཐར་གྱིས་པའི་མངོན་པར་རྟོགས་པ་འཆད་དེ། རྩེ་མོའི་མངོན་པར་རྟོགས་པ་ཐོབ་པ་སོ་སོ་བ་སེམས་བསྐྱེད་སོགས་མཁྱེན་གསུམ་གྱི་ཆོས་སུམ་ཅུ་པོ་རྣམས་དང་བསྡུས་པ་རྣམ་ཀུན་དང་རྩེ་སྦྱོར་ཉིད་དུ་རྟོགས་པའི་དོན་རྣམས་སྔར་རྒྱུད་ལ་བསྐྱེད་པའི་རིམ་པ་ལྟར་གོ་རིམ་བཞིན་དུ་བཀླགས་ནས་བརྟན་པོར་བྱ་བའི་ཕྱིར་རྣམ་པར་སྒོམ་པར་བྱེད་དགོས་པས་སོ།།གཉིས་པ་ ( མཚོན་བྱེད་ཀྱི་ཆོས་རྒྱས་པར་བཤད་ ) ལ། རྩ་བ། འགྲེལ་པ་གཉིས་ལས། དང་པོ་ནི།རྩ་བ། སྦྱིན་པ་ཤེས་རབ་བར་དག་དང༌།།སངས་རྒྱས་ལ་སོགས་དྲན་དང་ཆོས།།དངོས་མེད་ངོ་བོ་ཉིད་ཀྱིས་དེ།།མཐར་གྱིས་པ་ཡི་བྱ་བར་བཞེད།། ༡ རྩ་བའི་སྦྱོར་ཊཱིཀ མཐར་གྱིས་པའི་སྦྱོར་བ་ཆོས་ཅན། ཁྱོད་ལ་དབྱེ་ན་བཅུ་གསུམ་ཡོད་དེ། སྦྱིན་པའི་ཕར་ཕྱིན་ནས་ཤེས་རབ་ཀྱི་ཕར་ཕྱིན་གྱི་བར་དག་ཏུ་ཕར་ཕྱིན་དྲུག་ཉམས་སུ་ལེན་པའི་མཐར་གྱིས་སྦྱོར་དྲུག་དང༌། སངས་རྒྱས་དང་ལ་སོགས་པ་ཆོས་དང་དགེ་འདུན་དང་ཚུལ་ཁྲིམས་དང་གཏོང་བ་དང་ལྷ་རྗེས་སུ་དྲན་པ་སྟེ་རྗེས་དྲན་དྲུག་གི་མཐར་གྱིས་སྦྱོར་དྲུག་དང༌། ཆོས་ཐམས་ཅད་དངོས་པོ་མེད་པའི་ངོ་བོ་ཉིད་དུ་རྟོགས་པའི་མཐར་གྱིས་སྦྱོར་གཅིག་སྟེ་ཉམས་ལེན་གྱི་བྱ་བ་དྲུག་དང་དྲུག་དང་གཅིག་ཏུ་བཞེད་པས་སོ།།ངག་དོན། མཐར་གྱིས་པའི་སྦྱོར་བ་ཆོས་ཅན།

བཅུ་གསུམ་ཡོད་དེ། ཕར་ཕྱིན་དྲུག་རྗེས་དྲན་དྲུག །དངོས་མེད་ངོ་བོ་ཉིད་ཀྱི་མཐར་གྱིས་སྦྱོར་རྣམས་སུ་དབྱེར་ཡོད་པའི་ཕྱིར། གཉིས་པ་ ( འགྲེལ་པ་ ) ལ་གཉིས། སོ་སོར་བཤད་པ་དང༌། བསྡུས་ཏེ་བསྟན་པའོ། །དང་པོ་ ( སོ་སོར་བཤད་པ་ ) ལ་གསུམ། སྦྱོར་བ་ཕུན་ཚོགས། བསམ་པ་ཕུན་ཚོགས། རྣམ་པར་དག་བྱེད་ཀྱི་ཆོས་སོ། །དང་པོ་ནི། འགྲེལ་པ། ཕ་རོལ་ཏུ་ཕྱིན་པ་བཞི་ནི་ཤེས་རབ་ཀྱི་ཕ་རོལ་ཏུ་ཕྱིན་པའི་ནང་དུ་འདུས་པའི་ཕྱིར། སྦྱིན་པའི་ཕ་རོལ་ཏུ་ཕྱིན་པ་ལ་སོགས་པ་འཁོར་གསུམ་རྣམ་པར་དག་པས་རབ་ཏུ་ཕྱེ་བ་དྲུག་རྣམ་པ་ཐམས་ཅད་དུ་ཡོངས་སུ་རྫོགས་པས་ས་བཅུ་ཡང་དག་པར་སྒྲུབ་པར་བྱེད་པ་དང༌།འགྲེལ་པའི་སྦྱོར་ཊཱིཀ སྦྱིན་པའི་ཕ་རོལ་ཏུ་ཕྱིན་པ་ལ་སོགས་པ་འཁོར་གསུམ་རྣམ་པར་དག་པས་རབ་ཏུ་ཕྱེ་བ་ཕ་རོལ་ཏུ་ཕྱིན་པ་དྲུག་རྣམ་པ་ཐམས་ཅད་དུ་ཡོངས་སུ་རྫོགས་པར་བྱེད་པའི་མཐར་གྱིས་སྦྱོར་བ་ཆོས་ཅན། ཁྱོད་བསྒོམས་པ་ལ་དགོས་པ་ཡོད་དེ། རབ་ཏུ་དགའ་བ་ལ་སོགས་པའི་ས་བཅུ་ཡང་དག་པར་སྒྲུབ་པར་བྱེད་པའི་ཆེད་ཡོད་པས་སོ། །ས་བཅུ་རྫོགས་པ་ལ་ཕར་ཕྱིན་བཅུ་དགོས་པ་མ་ཡིན་ནམ་ཞེ་ན། སྐྱོན་མེད་དེ། ཕ་རོལ་ཏུ་ཕྱིན་པ་ཕྱི་མ་བཞི་ནི་ཤེས་རབ་ཀྱི་ཕ་རོལ་ཏུ་ཕྱིན་པའི་ནང་དུ་འདུས་པའི་ཕྱིར། གཉིས་པ་ ( བསམ་པ་ཕུན་ཚོགས་ ) ལ། ལམ་གྱི་རྟེན་རྗེས་སུ་དྲན་པ། ལམ་གྱི་གཞི་རྗེས་སུ་དྲན་པ། ལམ་གྱི་དཔང་པོ་རྗེས་སུ་དྲན་པ་དང་གསུམ་ལས། དང་པོ་ལམ་གྱི་རྟེན་ནི་དཀོན་མཆོག་གསུམ་སྟེ། དང་པོ་སངས་རྒྱས་རྗེས་སུ་དྲན་པ་ནི། འགྲེལ་པ། དྲན་པ་ཉེ་བར་བཞག་པ་ལ་སོགས་པ་དང༌། བྱང་ཆུབ་ཀྱི་ཡན་ལག་གི་རྣམ་པ་དང༌། འཕགས་པའི་ལམ་ཡན་ལག་བརྒྱད་པ་སྟེ། དོན་དམ་པར་དྲན་པ་མེད་པའི་མཚན་ཉིད་ཀྱི་སངས་རྒྱས་རྗེས་སུ་དྲན་པ་རྣམ་པ་གསུམ་པོ་གོ་རིམ་བཞིན་དུ་ངེས་པར་འབྱེད་པའི་ཆ་དང་མཐུན་པ་དང་མཐོང་བ་དང་སྒོམ་པའི་ལམ་བརྗོད་པར་བྱེད་པ་དང༌། འགྲེལ་པའི་སྦྱོར་ཊཱིཀ ལམ་གསུམ་རྗེས་སུ་དྲན་པ་ཡིན་ཞིང་ལམ་གསུམ་ནི་གོ་རིམ་བཞིན་དུ་ངེས་པར་འབྱེད་པའི་ཆ་དང་མཐུན་པ་རྗོད་པར་བྱེད་པ་དྲན་པ་ཉེ་བར་གཞག་པ་ལ་སོགས་ཀྱི་མིང་གིས་བསྟན་པ་དང༌།

མཐོང་བའི་ལམ་རྗོད་པར་བྱེད་པ་བྱང་ཆུབ་ཀྱི་ཡན་ལག་གི་རྣམ་པ་ཞེས་པས་བསྟན་པ་དང༌། སྒོམ་པའི་ལམ་རྗོད་པར་བྱེད་པ་འཕགས་པའི་ལམ་ཡན་ལག་བརྒྱད་པ་ཞེས་པས་བསྟན་པ་སྟེ་དེ་ལྟ་བུའི་ལམ་གསུམ་དོན་དམ་པར་དྲན་པ་མེད་ཅིང་ཀུན་རྫོབ་ཏུ་མཚན་ཉིད་ཀྱི་སྒོ་ནས་རྗེས་སུ་དྲན་པའི་སངས་རྒྱས་རྗེས་སུ་དྲན་པ་ནི་ཆོས་ཅན། རྣམ་པ་གསུམ་ཡོད་དེ། ལམ་གསུམ་རྗེས་སུ་དྲན་པ་གསུམ་ཡོད་པས་སོ།།དེ་ཡང་སངས་རྒྱས་ནི་རྒྱུ་འདི་དག་ལས་སྐྱེས་སོ་ཞེས་རྒྱུའི་སྒོ་ནས་རྗེས་སུ་དྲན་པའོ།།གཉིས་པ་ཆོས་རྗེས་དྲན་ནི། འགྲེལ་པ། དེ་བཞིན་དུ་དགེ་བ་དང་མི་དགེ་བ་དང་ལུང་དུ་མ་བསྟན་པའི་ཆོས་རྗེས་སུ་དྲན་པ་དང༌།འགྲེལ་པའི་སྦྱོར་ཊཱིཀ སྔ་མ་དེ་བཞིན་དུ་དགེ་བ་དང་མི་དགེ་བ་དང་ལུང་དུ་མ་བསྟན་པའི་ཆོས་རྣམས་དོན་དམ་པར་དྲན་པ་མེད་ཅིང་ཀུན་རྫོབ་ཏུ་མཚན་ཉིད་ཀྱི་སྒོ་ནས་རྗེས་སུ་དྲན་པ་ཆོས་ཅན། ཆོས་རྗེས་དྲན་གྱི་ངོ་བོ་ཡིན་ཏེ། དེའི་དོན་ལྡོག་ཡིན་པས་སོ།།གསུམ་པ་དགེ་འདུན་རྗེས་དྲན་ནི། འགྲེལ་པ། སྔ་མ་བཞིན་དུ་བྱང་ཆུབ་སེམས་དཔའ་འཕགས་པ་ཕྱིར་མི་ལྡོག་པའི་དགེ་འདུན་རྗེས་སུ་དྲན་པ་དང༌། འགྲེལ་པའི་སྦྱོར་ཊཱིཀ སྔ་མ་དེ་བཞིན་དུ་བྱང་ཆུབ་སེམས་དཔའ་འཕགས་པ་ཕྱིར་མི་ལྡོག་པའི་དགེ་འདུན་རྣམས་དོན་དམ་པར་དྲན་པ་མེད་ཅིང་ཀུན་རྫོབ་ཏུ་མཚན་ཉིད་ཀྱི་སྒོ་ནས་རྗེས་སུ་དྲན་པ་ཆོས་ཅན། དགེ་འདུན་རྗེས་སུ་དྲན་པའི་ངོ་བོ་སོགས་སོ།།གཉིས་པ་ལམ་གྱི་གཞི་རྗེས་སུ་དྲན་པ་ནི། འགྲེལ་པ། དེ་བཞིན་དུ་ཚུལ་ཁྲིམས་དང༌། གཏོང་བ་དང༌།འགྲེལ་པའི་སྦྱོར་ཊཱིཀ ཚུལ་ཁྲིམས་གསུམ་སྔར་བཤད་པ་དེ་བཞིན་དུ་དོན་དམ་པར་དྲན་པ་མེད་ཅིང་ཀུན་རྗོབ་ཏུ་མཚན་ཉིད་ཀྱི་སྒོ་ནས་སོགས་དང༌། གཏོང་བ་གསུམ་པོ་སྔར་བཤད་པ་རྣམས་དང་མཐུན་པར་རྗེས་སུ་དྲན་པ་སོགས་དཔགས་ཏེ་སྦྱོར་བ་བཀོད་དོ།།གསུམ་པ་ལམ་གྱི་དཔང་པོ་རྗེས་སུ་དྲན་པ་ནི། འགྲེལ་པ། ལྷ་རྗེས་སུ་དྲན་པ་དང༌།

འགྲེལ་པའི་སྦྱོར་ཊཱིཀ ལྷར་སྐྱེས་པའི་འཕགས་པ་རྗེས་སུ་དྲན་ཚུལ་དང་དོན་སྦྱོར་རིགས་འདྲའོ།།གསུམ་པ་རྣམ་པར་དག་བྱེད་ཀྱི་ཆོས་ནི། འགྲེལ་པ། གཟུགས་ལ་སོགས་པའི་ཆོས་ཐམས་ཅད་དངོས་པོ་མེད་པའི་ངོ་བོ་ཉིད་དུ་ཁོང་དུ་ཆུད་པས་རྟོགས་པ། འགྲེལ་པའི་སྦྱོར་ཊཱིཀ གཟུགས་ལ་སོགས་པའི་ཆོས་ཐམས་ཅད་བདེན་པའི་དངོས་པོ་མེད་པའི་ངོ་བོ་ཉིད་དུ་ཁོང་དུ་ཆུད་པས་རྟོགས་པའི་སྦྱོར་བ་ཆོས་ཅན། དངོས་མེད་ངོ་བོ་ཉིད་ཀྱི་མཐར་གྱིས་སྦྱོར་བའི་ངོ་བོ་ཡིན་ཏེ། དེའི་དོན་ལྡོག་ཡིན་པས་སོ།།གཉིས་པ་བསྡུས་ཏེ་བསྟན་པ་ནི། འགྲེལ་པ། གང་ཡིན་པ་དེ་ནི་འདིར་མཐར་གྱིས་པའི་བྱ་བར་བཞེད་པ་ཡིན་ནོ།།འགྲེལ་པའི་སྦྱོར་ཊཱིཀ དེ་ལྟར་བསྟན་པའི་བཅུ་གསུམ་པོ་གང་ཡིན་པ་དེ་ནི་ཆོས་ཅན། སྐབས་འདིར་མཐར་གྱིས་པའི་སྦྱོར་བའི་ཉམས་སུ་བླང་བར་བྱ་བར་བཞེད་པ་ཡིན་ནོ་སྟེ། དེའི་བསྒོམ་བྱ་ཡིན་པས་སོ།།གསུམ་པ་སྐབས་ཀྱི་གྲངས་བརྗོད་པས་མཇུག་བསྡུ་བ་ནི། འགྲེལ་པ། ཤེས་རབ་ཀྱི་ཕ་རོལ་ཏུ་ཕྱིན་པའི་མན་ངག་གི་བསྟན་བཅོས་མངོན་པར་རྟོགས་པའི་རྒྱན་ཅེས་བྱ་བ་ལས་སྐབས་དྲུག་པའི་འགྲེལ་པའོ།། །།འགྲེལ་པའི་སྦྱོར་ཊཱིཀ ཤེས་རབ་ཀྱི་ཕ་རོལ་ཏུ་ཕྱིན་པའི་མན་ངག་གི་བསྟན་བཅོས་མངོན་པར་རྟོགས་པའི་རྒྱན་ཞེས་བྱ་བ་རྩ་འགྲེལ་གྱི་སྦྱོར་བའི་ཊཱི་ཀ་ལས། སྐབས་དྲུག་པའི་རྒྱན་གཞུང་འགྲེལ་པ་དང་བཅས་པའི་ཚིག་དོན་གྱི་སྦྱོར་ངག་བསྟན་ཟིན་པའོ།། །།སྐབས་དྲུག་པ། མཐར་གྱིས་སྦྱོར་བའི་སྦྱོར་ཊཱིཀམངོན་རྟོགས་རྒྱན་གྱི་སྦྱོར་ཊཱིཀ་འཇིག་རྟེན་གསུམ་སྒྲོན་ལ་འཇུག་པ།
